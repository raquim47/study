01. 변수 : 데이터 저장

{
    // 변수(variables)는 변하는 데이터(값)을 저장할 수 있는 메모리 공간        
    // 변수를 선언할 때에는 var 키워드를 변수명 앞에 붙인다.
    // 변수명은 한글 사용X / 영문 / 숫자 / 일부 특수 문자(_,$)만 가능 변수를 선언하면 변수명으로 데이터를 저장할 수 있는 공간이 생성.
    // 변수에 저장할 수 있는 데이터의 종류 : 문자형(Srting), 숫자형(Number), 논리형(Boolean), 빈(Null) 데이터

    var x = 1;            
    var y = 369;            
    var z = "seongjin";  
            
    document.write(x);
    document.write(y);
    document.write(z);
}

04. 변수 : 변수의 종류 : 지역변수 + 전역변수

{
    // var의 범위 : function
    // let의 범위 : 스코프

    let x = 100;    //전역변수
    let y = 200;    //전역변수

    function func(){
        let x = 100;        //지역변수
        let z = "javascript";  //지역변수
        x = 200;           //지역변수 100->200
        y = 300;           //전역변수 200->300      

        document.write(x);
        document.write(y);
        document.write(z);
    }
    func();

    document.write(x);
    document.write(y);
    document.write(z);
}

17. 객체 : 데이터를 저장(키와 값) : 표현 방법8 : 객체 속에 함수 

{
    const obj = {
        a: 100,
        b: [200, 300],
        c: {x: 400, y: 500},
        d: "javascript",
        e: function(){
          return "안녕하세요";
        },
        f: function(){
          return `안녕하세요 ${obj.d} 입니다`;
        },
        g: function(){
          return "안녕하세요 " + this.d + " 입니다";
        }
    }

    document.write(obj.a);
    document.write(obj.b);
    document.write(obj.b[0]);
    document.write(obj.b[1]);
    document.write(obj.c.x);
    document.write(obj.c.y);
    obj.e();
    obj.f();
    obj.g();
}
