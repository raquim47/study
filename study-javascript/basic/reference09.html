정규식

정규 표현식, 또는 정규식은 문자열에서 특정 문자 조합을 찾기 위한 패턴. 즉, 문자를 검사하고 싶을 때 사용하는 식. 사용 예시) /찾고자하는 패턴/

연습해볼 수 있는 사이트 : https://regexr.com/5mhou

그룹과 범위

| : 또는

 ex) /Hi|Hello/

() : 그룹

 ex) /(Hi|Hello)/

[] : 문자셋, 괄호안의 어떤 문자든

 ex) /[abc]/

[^] : 부정 문자셋, 괄호안의 어떤 문자도 포함되지 않는

 ex) /[^abc]/

찾지만 기억(그룹)하지 않음

 ex) /(?:Hi|Hello)/

수량자

? : 없거나 있거나(zero or one)

 ex) /gra?y/ : a가 없거나 있는 문자 모두

* : 없거나 있거나 많거나(zero or more)

 ex) /gra*y/ : 0, 1, 1~

+ : 있거나 많거나(one or more)

 ex) /gra+y/ : 1, 1~

[n] : n번 반복

 ex) /gra{3}y/ : a가 3번 나오는
          /gra{3,}y/ : a가 최소 3번 나오는
          /gra{3, 5}y/ : a가 최소 3번 최대 5번 나오는

문자 클래스

\b : 단어 경계(앞에 붙이면 단어 앞에 시작하는 것, 뒤에 붙이면 끝에 있는 것)

\B : 단어 경계가 아님(앞에 붙이면 단어 앞에 시작하지 않는 것, 뒤에 붙이면 끝에 있지 않은 것)

^ : 문장의 시작

 ex) /^car/

$ : 문장의 끝

 ex) /car$/

\ : 특수 문자가 아닌 문자를 찾을 때

 ex) /\./ : .찾기

. : 모든 문자

 ex) /\./ : .찾기

\d, \D : 숫자, 숫자가 아닌 것

\w, \W : 문자, 문자가 아닌 것

\s, \S : 공백, 공백이 아닌 것(모든 문자)

그밖에 자주 쓰는 것

[a-zA-Z] : 아무 알파벳

[ㄱ-ㅎ가-힣ㅏ-ㅣ] : 아무 한글

[a-zA-Z0-9ㄱ-ㅎ가-힣ㅏ-ㅣ] : 알파벳, 한글, 숫자

유튜브 링크 아이디 가져오기

const regex = /(?:https?:\/\/)?(?:www\.)?youtu.be\/([a-zA-Z0-9-]{11})/;
          const url = 'http://youtu.be/-ZClicWm0zM';
          const result = url.match(regex);
          
          result[1];
          결과 : '-ZClicWm0zM'
